# Personal Portfolio (Vite + React + Tailwind)

This is a starter personal portfolio built with React (Vite), Tailwind CSS and Framer Motion.

Features:
- Responsive, mobile-first design
- Dark mode
- Smooth scrolling and active-section highlighting
- Typing animation in hero
- Reusable components
- Accessible form and ARIA attributes

Quick start:

1. Install dependencies

```bash
cd /Users/Maan/Desktop/\"Muhammad Usman\"
npm install
```

2. Run dev server

```bash
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

Notes:
- The contact form is simulated; wire it to your backend or service provider.
- Update `src/data` with your real projects and experience.
